# bilibili_like_all_dynamic

烂活， 点赞UP的所有动态。

你知道我要做什么😇😇😇


# 使用

1. 修改`app.js`中的配置

填入自己账号的cookie, 以及待点赞UP的uid

```js
let config = {
    cookie: {
        sid: '',
        DedeUserID: '',
        DedeUserID__ckMd5: '',
        SESSDATA: '',
        bili_jct: '',
    },
    like_uid: 672328094
}
```

2. 运行


```bash
npm install
npm run
```
